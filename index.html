Array

  1. 方法名： push

    作用：往数组的最后一位添加数据

    返回值： push后数组的长度

    EG:

    var arr=[1,2,3,4,5];arr.push(8);

    return: 6;

    原数组是否发生改变：是

    方法调用后原数组为：[1,2,3,4,5,8];

  2. 方法名： unshift

    作用：往数组的最前面添加数据

    返回值： unshift后数组的长度

    EG:

    var arr=[1,2,3,4,5];arr.unshift(8);

    return: 6;

    原数组是否发生改变：是

    方法调用后原数组为：[8,1,2,3,4,5];

  3. 方法名： pop

    作用：删除数组的最后一个

    返回值： 删除的值

    EG:

    var arr=[1,2,3,4,5];arr.pop();

    return: 5;

    原数组是否发生改变：是

    方法调用后原数组为：[1,2,3,4]; //如果数组里没有东西了继续删除会返回undefined

  4. 方法名： shift

    作用：删除数组的第一个

    返回值： 删除的值

    EG:

    var arr=[1,2,3,4,5];arr.shift();

    return: 1;

    原数组是否发生改变：是

    方法调用后原数组为：[1,2,3,4];

  5. 方法名： splice

    作用：向数组中间某个位置添加内容从索引n开始，删除m个内容，把新增的内容x放在n的前面

    返回值： 删除的值（数组）

    EG:

    var arr=[1,2,3,4,5,6];arr.splice(1,2,8,8);

    return: [2, 3];

    原数组是否发生改变：是

    方法调用后原数组为：[1, 8, 8, 4, 5, 6];

    splice(0,0,x);相当于unshift

    splice(ary.length,0,x);相当于push

  6. 方法名： reverse

    作用：数组倒过来排列

    返回值： 删除的值（数组）

    EG:

    var arr=[1,2,3,4,5,6];arr.reverse();

    return: [6, 5, 4, 3, 2, 1];

    原数组是否发生改变：是

    方法调用后原数组为：[6, 5, 4, 3, 2, 1];

  7. 方法名： join

    作用：把数组用指定的方式连接转成字符串

    返回值： 按规则拼接后的字符串

    EG:

    var arr=[1,2,3,4,5,6];arr.join();

    return: "1,2,3,4,5,6";//默认逗号

    原数组是否发生改变：否

    方法调用后原数组为：[1,2,3,4,5,6];

    var arr=[1,2,3,4,5,6];arr.join(""); return: "123456";

    var arr=[1,2,3,4,5,6];arr.join("*"); return: "1*2*3*4*5*6";

  8. 方法名： sort

    作用：数组排序（只能比较10以内的数）

    返回值： 按规则排序后的数组

    EG:

    var arr=[3,34,1,12,457,1232,45,23,68,13,32];arr.sort();

    return: [1, 12, 1232, 13, 23, 3, 32, 34, 45, 457, 68];//只能比较10以内的数字

    原数组是否发生改变：是

    方法调用后原数组为：[1, 12, 1232, 13, 23, 3, 32, 34, 45, 457, 68];

    var arr=[4,1,7,2,4,8,9,3];arr.sort() return: [1, 2, 3, 4, 4, 7, 8, 9];

    var arr=[3,34,1,12,457,1232,45,23,68,13,32];arr.sort(function(a,b){return b-a});// b-a降序 a-b升序 return: [1232, 457, 68, 45, 34, 32, 23, 13, 12, 3, 1];

  9. 方法名： concat

    作用：数组连接

    返回值： 连接后的数组

    EG:

    var arr=[1,2,3,4];var arr1=[7,8,9];arr.concat(arr1);

    return: [1, 2, 3, 4, 7, 8, 9];//只能比较10以内的数字

    原数组是否发生改变：否

    方法调用后原数组为：[1, 2, 3, 4];

  10. 方法名： toString

    作用：把数组转成字符串用逗号隔开

    返回值： 连接后的数组

    EG:

    var arr=[1,2,3,4];arr.toString();

    return: "1,2,3,4";//只能比较10以内的数字

    原数组是否发生改变：否

    方法调用后原数组为：[1, 2, 3, 4];

  11. 方法名： slice

    作用：截取数组（n,m]位的值

    返回值： 被选中的值

    EG:

    var arr=[1,2,3,4];arr.slice(1,3);

    return: [2,3];//只能比较10以内的数字

    原数组是否发生改变：否

    方法调用后原数组为：[1, 2, 3, 4];

    var arr=[1,2,3,4];arr.slice(1); return: [2,3,4];

    var arr=[1,2,3,4];arr.slice(0); return: [1,2,3,4];//复制数组

  12. 方法名： indexOf

    作用：在数组中查找某一个字符串出现的索引位置

    返回值： 索引值

    EG:

    var arr=[1,2,3,4];arr.indexOf(3);

    return: 2;//只能比较10以内的数字

    原数组是否发生改变：否

    方法调用后原数组为：[1, 2, 3, 4];//IE9+

  13. 方法名： forEach

    作用：为数组中的每个元素执行指定操作。

    返回值： 无

    EG:

    var arr=[1,32,4,5,6];arr.forEach(function(value){console.log(value)});

    return: undefined;//只能比较10以内的数字

    原数组是否发生改变：否

    方法调用后原数组为：[1,32,4,5,6];//IE9+

  14. 方法名： map

    作用：为数组中的每个元素执行指定操作。

    返回值： 根据函数的返回值来定

    EG:

    var arr=[1,32,4,5,6];arr.map(function(value,index){return value*5});

    return: [5, 160, 20, 25, 30];

    原数组是否发生改变：否

    方法调用后原数组为：[1,32,4,5,6];//IE9+
